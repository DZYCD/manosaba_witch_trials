# 🌙 魔女裁决 - 魔法少女的推理游戏

基于《魔法少女的魔女审判》的二创同人游戏。

## 🎮 游戏介绍

玩家扮演樱羽艾玛，与12位AI魔法少女一起进行推理讨论，找出杀害莲见蕾雅的真凶。

## 🚀 如何游玩

1. 访问游戏页面
2. 配置你的 OpenAI API（支持兼容接口）
3. 开始新游戏，参与讨论
4. 在30分钟内找出真凶并投票

## ⚙️ API 配置

游戏需要 OpenAI 兼容的 API：
- API URL：如 `https://api.openai.com/v1/chat/completions`
- API Key：你的密钥
- 模型：推荐 `gpt-4o-mini` 或更高

API 配置保存在浏览器 Cookie 中，不会上传到任何服务器。
